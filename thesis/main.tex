\section{Background}
Many real-world routing and scheduling problems are subject to uncertainty. Take delivery service as an example, a courier has several customers located at different cities and it needs to plan a route that visits the customers. The delivery person may find that 
a household might not have packages to deliver today, or customer requests can be canceled, etc..
A classical abstraction is the stochastic
traveling salesperson problem (stochastic TSP), where each vertex $v$ comes with a specified probability $p_v$ and becomes \textit{active} (requires service) independently with probability $p_v$. The goal is to find a tour that visits all active vertices and minimizes the expected travel cost, where the cost comes from the underlying metric on vertices.

An important modeling choice is whether the salesperson is allowed to adapt their future decisions based on information revealed during the execution of the tour. In the nonadaptive (a priori) setting, the salesperson commits to a master tour that only depends on the underlying metric and activation probabilities and not the actual realizations. During the actual execution of the tour, i.e., once the set of active vertices is realized, the salesperson follows the tour while shortcutting inactive vertices. In the adaptive setting, decisions may depend on observations revealed along the way, allowing the tour to be adjusted on the fly. 

A fundamental analysis question is how much adaptivity can improve performance relative to an optimal nonadaptive tour.
A related algorithmic question is whether we can design provably good adaptive policies using limited computational resources.
Note that computing an optimal adaptive policy (or a nonadaptive master tour) for stochastic TSP is computationally hard, since the setting with $p_v = 1$ for all $v \in V$ corresponds to the classical TSP which is known to be APX-hard\cite{Karpinski2015}. 
Also, describing an optimal adaptive policy might require exponential space. 
Thus, we will be interested in designing adaptive policies that can be implemented efficiently 
and its expected cost can be provably bounded in terms of the optimal cost within some small factor $\alpha \geq 1$.  

\section{Notations and Definitions}
Let $(\{r\}\cup V,d)$ be a finite metric space on $n+1$ vertices (a depot vertex $r$ and $n$ clients) equipped with a distance function $d$. 
That is, $d$ satisfies the triangle inequality: for any three vertices $u,v,w \in V$, $d(u,v) \leq d(u,w) + d(w,v)$ holds. 
A metric is said to be \emph{symmetric} if $d(x, y) = d(y, x)$ for any $x, y$.

For classical TSP, we want to find a tour $T = (r,v_1,v_2,\ldots,v_n,r)$ that starts and ends at the depot, visits all vertices exactly once, and minimizes the total distance traversed, $c(T) := d(r,v_1) + d(v_1,v_2) + \cdots + d(v_{n-1},v_n) + d(v_n,r)$. 

In the stochastic setting, the set of active vertices is a subset $A \subseteq V$ generated by independent activations: each vertex $v \in V$ is active (i.e., $v \in A$) independently with probability $p_v$.
% It precisely models a delivery planning the delivery route for customers located at different places and each customer needs delivery with a specified probability independently.
% 
We consider three different benchmarks for stochastic TSP depending on how much the tour is allowed to depend on the realization of the set of active vertices:
\begin{enumerate}
    \item \textbf{A posteriori TSP:} %\texorpdfstring{\cite{Shmoys2008}}{}} 
        Here, the salesperson knows the set $A$ of active vertices. 
        Let $\opt(A)$ denote the cost of an optimal TSP tour that visits $\{r\} \cup A$. (Note that this is simply an instance of the classical TSP.) 
        The cost of a posteriori TSP is defined to be 
        $\optpost := \E_{A \sim \mu} [\opt(A)]$, where $\mu$ is the probability distribution over the set of active vertices. 
        Note $\mu(A) = \Pi_{v \in A}p_v \cdot \Pi_{v \notin A}(1 - p_v)$ for any $A \subseteq V$.  
        
    \item \textbf{A priori (nonadaptive) TSP:} 
        Here, the salesperson only knows the metric and the activation probabilities and they are required to compute a master tour $T = (r, v_1,\ldots,v_n,r)$ that visits all vertices while starting and ending at $r$. 
        For any set $A$ of active vertices, the tour $T\vert_A$ traversed by the salesperson (for that $A$) is obtained by shortcutting $T$ to include only vertices in $A$. 
        More precisely, if $1 \leq i_1 < i_2 < \cdots < i_{|A|} \leq n$ are such that $\{v_{i_k} : 1 \leq k \leq |A|\} = A$, then $T\vert_A = (r,v_{i_1},v_{i_2},\ldots,v_{i_{|A|}},r)$.  
        We overload the notation $c$ and define the cost of $T$ to be $\E_{A \sim \mu} [c(T\vert_A)]$. 
        We define $\optprior$ to be the cost of an optimal master tour.  
        
    \item \textbf{Adaptive TSP:} 
        In this case the salesperson knows all the activation probabilities but not the realization of the set of active vertices $A$. 
        However, on the day of the execution of the tour, the salesperson calls all the customers one by one in some order. 
        If the customer $v$ that they call is in $A$, then the salesperson \emph{must} visit $v$ right away from their current position before calling any other customer. 
        On the other hand, if $v$ is not in $A$, then the salesperson remains at the current position. 
        In an adaptive policy, the salesperson is allowed to use the revealed information about the customers that they have called to decide which customer to call next.    
        
        Formally, the decision tree of an adaptive policy can be represented as a function $\pi: (\{r\} \cup V) \times 2^V \rightarrow \{r\} \cup V \cup \{\bot\}$, where $\pi(u,S)$ denotes the vertex that should be called next given that the salesperson is currently at $u$ and the set of vertices that have never been called is exactly $S$. 
        We require that $\pi$ is a well-defined policy.
        That is: (a)~$\pi(u,S) := \bot$ whenever $u \in S$, which models invalid input to the policy; (b)~$\pi(u,S) = r$ iff $S = \emptyset$, which models that the salesperson must return to the depot only when there are no other vertices to call; and (c)~$\pi(u,S) \in S$ iff $S \neq \emptyset$, which models that the salesperson must call one of the vertices that have not been called yet. 

        We now describe the (random) tour $T(A;\pi)$ traversed by an adaptive policy $\pi$ when the set of active vertices is $A$.  
        We drop the argument $\pi$ when it is clear from the context. 
        Note that $A$ is unknown to $\pi$. 
        Initially, $u \leftarrow r$ and $S \leftarrow V$, which models that the salesperson starts at the depot vertex and they have not called any vertex so far.  
        As long as $S \neq \emptyset$ or $u \neq r$, the salesperson computes $v = \pi(u,S)$. 
        If $S \neq \emptyset$, the salesperson calls $v$. 
        (Note that $v \in S$ since $\pi$ is well-defined.) 
        If $v \in A$, then the salesperson travels from $u$ to $v$. 
        Otherwise, the salesperson remains at $u$. 
        In the former case, we update $u \leftarrow v$.
        In both cases, we update $S \leftarrow S \setminus \{v\}$.
        If $S = \emptyset$, then $v = r$ holds.
        As all vertices have been called by now, the salesperson simply returns to $r$. 
        This is modeled by setting $u \leftarrow r$.
        
        Observe that $T(A;\pi)$ corresponds to a random tour $(r,v_1,\ldots,v_{|A|},r)$ on $\{r\} \cup A$, where $v_i$ is the $i$th among $A$ to be called. 
        We again overload the cost function and define $c(\pi) := \E_{A \sim \mu}[c(T(A;\pi))]$.   
        
\end{enumerate}

\section{Research Question}
A priori TSP is the most nonadaptive policy, since we cannot adjust our strategy at all.
A posteriori TSP is the most adaptive policy where we can always know which vertices would be active and then make our decision.
Adaptive TSP lies in between, because we are allowed to use the revealed information about the vertices that they have called to adjust the next decision.

We define $\mathcal{I} := (V \cup \{r\}, d, \{p_v\})$ to be an instance of a stochastic TSP then clearly for any instance $\mathcal{I}$ we have:
\begin{equation}\label{eq:optineq}
\optpost \le \optadapt \le \optprior    
\end{equation}
since from left to right we have less adaptivity and information, thus end up getting worse optimal cost.

We define \emph{adaptivity gap} to be $\sup_{\mathcal{I}} \f{\optprior}{\optadapt}$, which measures the worst-case ratio over all instances of problem between the cost of an optimal a priori solution vs. an optimal adaptive policy. We also define \emph{clairvoyance gap} to be $\sup_{\mathcal{I}} \f{\optadapt}{\optpost}$ and \emph{obliviousness gap} to be $\sup_{\mathcal{I}} \f{\optprior}{\optpost}$. The clairvoyance gap measures the benefits that an a posteriori solution can get by having clairvoyance, i.e., knowledge of the future. The obliviousness gap measures the gap between being completely oblivious (nonadaptive) to the realizations vs. the a posteriori solution. We remark that literature on a priori TSP uses the term adaptivity gap for the obliviousness gap that we defined above. We deviate from the terminology used in the literature \cite{Christalla2025} for the sake of clarity. The terminology that we use is more standard in stochastic optimization.

For symmetric metrics, the obliviousness gap is known to be at most $O(1)$; in particular obliviousness gap $\le 3$ is known in \cite{Shmoys2008}. 
The same upper bound carries over to clairvoyance gap and adaptivity gap because of Eq~\eqref{eq:optineq}. 
One of the objectives of this project is to obtain better upper and lower bound on clairvoyance gap and adaptivity gap.
% note that the lower bound of prior/post is not clear (to show that prior solution cannot not be very good to some extent). in paper "Improved guarantees for the a priori TSP" we have lower bound for master tour ratio, which means if we use algorithm with this pattern: sample a set, construct best tsp, then connect edges; this scheme will give you no good than 2.6** than optimal a priori in some instance

For asymmetric metrics, Christalla, Puhlmann and Traub \cite{Christalla2025} show that the obliviousness gap is known to be $O(\sqrt{n})$ and $\Omega(n^{1/4} \cdot \log^{-1}n)$. 
Moreover, it is not hard to show that the construction in Section~3.1 in \cite{Christalla2025} also implies that adaptivity gap = $\Omega(n^{1/4} \cdot \log^{-1}n)$. 
Another key objective of my project is to prove or disprove that the clairvoyance gap for asymmetric metrics is $O(1)$. In case of a positive resolution, a further goal would be to design an adaptive policy matching this gap up to constant factors. If not, my aim is to construct an infinite family of instances for which the clairvoyance gap grows to infinity as $n$ goes to infinity.

\section{Proof of epsilon normalization equivalence}
We call an instance $I = (V, c, p)$ $\varepsilon$-normalized if there exists a depot $d \in V$ with $p(d) = 1$ and $p(v) = \varepsilon$ for all $v \in V \setminus \{d\}$.

\begin{lemma}\label{lem:transform}
    There is a transform $g$ mapping any instance $I$ to an $\varepsilon$-normalized instance $I' = g(I)$ such that the following hold:
    \begin{alignat*}{2}
        (1-\xi)\optpost(I)  &\le{}& \optpost(I')  &\le{} \optpost(I) \\
        (1-\xi)\optprior(I) &\le{}& \optprior(I') &\le{} \optprior(I) \\
        (1-\xi)\optadapt(I) &\le{}& \optadapt(I') &\le{} \optadapt(I)
    \end{alignat*}
\end{lemma}

There is a lemma from paper \cite{Blauth2024} Lemma 32 which says:

\begin{lemma}[Existence of proper parameters]\label{lem:param}
    For  $\sigma \in (0, 1), \lambda > 0$ and $p \in (0, 1]$. There is $\varepsilon_p \in (0, 1)$ such that for any $\varepsilon \in (0, \varepsilon_p ]$ there is $k \in \mathbb{N}$ such that 
    \begin{enumerate}
        \item $1 - (1-\varepsilon)^k \le p$
        \item $(1-\lambda)p \le 1-(1-\varepsilon)^k$
    \end{enumerate}
\end{lemma}

\begin{proof}[Proof of Lemma \ref{lem:param}]
    We can simplify the inequalities into $1 - p \le (1-\varepsilon)^k \le 1 - (1-\lambda)p$. Suppose $p < 1$, then for any certain $\varepsilon$, the inequalities are equivalent to $\frac{\ln (1 - (1-\lambda)p)}{\ln (1 - \varepsilon)} \le k \le \frac{\ln (1 - p)}{\ln (1 - \varepsilon)}$. Since $k$ has to be an integer, we can choose $\varepsilon_p$ to be small enough such that $\frac{\ln (1 - p)}{\ln (1 - \varepsilon_p)} - \frac{\ln (1 - (1-\lambda)p)}{\ln (1 - \varepsilon_p)} > 1$. Suppose $p = 1$, then we only need to choose $k$ to be large enough such that $(1-\epsilon)^k \le \lambda$. Thus in this case $\varepsilon_p$ can be any value in $(0,1)$.
\end{proof}

With Lemma \ref{lem:param}, we can now prove Lemma \ref{lem:transform}.

\begin{proof}[Proof of Lemma \ref{lem:transform}]
    We firstly give a transform $g$ and then prove the required properties hold.

    For a certain instance $I = (V, c, p)$, we apply Lemma \ref{lem:param} for every $p(v), v \in V$ and get $\varepsilon_{p(v)}$. Then we pick $\varepsilon := \min \{\varepsilon_{p(v)}\ \mid v \in V\}$. Again by Lemma \ref{lem:param}, for each vertex $v$, there exists $k_v$ satisfying the required two inequalities. 
    
    The transform $g$ will split each vertex $v$ to $k_v$ copies with activation probability to be $\varepsilon$ and keep the original distance metric unchanged. Formally, suppose $g(I) = I' = (V', c', p')$, then $V' = \bigcup_{v \in  V} \{v_1, v_2, \dots, v_{k_v}\}$ where $v_1, v_2, \dots, v_{k_v}$ represents the copies of a vertex $v$. And we let the distance $c'(v_i, v'_j) = c(v, v')$ if $v \neq v'$ and $c'(v_i, v'_j) = 0$ if $v = v'$. We let the activation probability $p'(v) = \varepsilon$ for any $v \in V'$.

    Notice that transform $g$ can be decomposed as $g = g_{|V|}  \circ \cdots g_2 \circ g_1$ where $g_i$ is the transform that keeps everything unchanged except splitting the $i$-th vertex $v$ into a set of vertices $s(v) = \{v_{1}, v_{2}, \cdots, v_{k_{v}}\}$ and adjusting the distance and activation probability accordingly. 

    We firstly prove for such transform $g_i$, transforming $I$ to $I' = g_i(I)$, we have:

    \begin{alignat*}{2}
        (1-\eta)\optpost(I)  &\le{}& \optpost(I')  &\le{} \optpost(I) \\
        (1-\eta)\optprior(I) &\le{}& \optprior(I') &\le{} \optprior(I) \\
        (1-\eta)\optadapt(I) &\le{}& \optadapt(I') &\le{} \optadapt(I)
    \end{alignat*}

    \newcommand{\bp}[0]{\bar{p}}
    \newcommand{\tsp}[0]{\text{TSP}}
    Denote $\bp(v) := 1 - (1 - \varepsilon)^{k_v}$. We can see that  $\bp(v)$ corresponds to the probability that at least one vertex in $s(v)$ is active. Let $\pi: 2^{V'} \rightarrow 2^{V}$ denote the map $\pi(A') = \{v \in V \mid A' \cap s(v) \neq \emptyset\}$ from active vertices set in $I'$ to corresponding active vertices set in $I$. Since we are only considering transform $g_i$ which only split the $i$-th vertex $v^*$, by definition we have that for any $A_0 \ni v^*$, 
    \begin{align*}
        \Pr_{A \sim \mu}[A = A_0] = \Pi_{v \in A_0 \setminus \{v^*\}} p(v) \cdot \Pi_{v \notin A_0 \setminus \{v^*\}}(1 - p(v)) \cdot p(v^*)
    \end{align*}
    and that
    \begin{align*}
        &\Pr_{A' \sim \mu'} [\pi(A') = A_0] \\
        =& \Pi_{v \in A_0 \setminus \{v^*\}} p(v) \cdot \Pi_{v \notin A_0 \setminus \{v^*\}}(1 - p(v)) \cdot \Pr[\text{at least one vertex in } s(v^*) \text{is active}] \\
        =& \Pi_{v \in A_0 \setminus \{v^*\}} p(v) \cdot \Pi_{v \notin A_0 \setminus \{v^*\}}(1 - p(v)) \cdot \bp(v^*)
    \end{align*}
    
    So we have when $A_0 \ni v^*$, 
    \begin{align*}
       (1-\lambda)\Pr_{A \sim \mu}[A = A_0] \le\Pr_{A' \sim \mu'}[\pi(A') = A_0] \le \Pr_{A \sim \mu}[A = A_0]
    \end{align*}

    And by similar derivation the inequality holds also when $v^* \notin A_0$.

    We now prove a helping claim that the cost of optimal TSP tour for $I'$ restricted on active vertices $A'$, denoted as $c'_\tsp(A')$, is the same as the cost of optimal TSP tour for $I$ restricted on active vertices $\pi(A')$, denoted as $c_\tsp(\pi(A'))$. This is because $A'$ can be seen as $\pi(A')$ together with some additional vertices so $c'_\tsp(A') \ge c_\tsp(\pi(A'))$; on the other hand the optimal TSP for $\pi(A')$ can be simply extended to a TSP tour for $A'$: instead of visit one vertex each time in $\pi(A')$, we visit all corresponding vertices in $A'$, and since the distance between vertices within $s(v)$ is $0$ so the extended tour for $A'$ has exactly the same cost of the optimal tour for $\pi(A')$, which means  $c'_\tsp(A') = c_\tsp(\pi(A'))$.

    Now consider that 
    $$
        \optpost(I) = \sum_{A \subset V} \Pr[A]\opt(A)
    $$ and that 
    \begin{align*}
        \optpost(I') &= \sum_{A' \subset  V'} \Pr[A']\opt(A') \\
        &= \sum_{A \subset V} \Pr_{A' \sim \mu'}[\pi(A')=A_0]\opt(A') \\
        &= \sum_{A \subset V} \Pr_{A' \sim \mu'}[\pi(A')=A_0]\opt(A)
    \end{align*}

    Since 
    \begin{align*}
       (1-\lambda)\Pr_{A \sim \mu}[A = A_0] \le\Pr_{A' \sim \mu'}[\pi(A') = A_0] \le \Pr_{A \sim \mu}[A = A_0]
    \end{align*}

    we finally have that
    $$
        (1-\lambda)\optpost(I) \le \optpost(I')  \le \optpost(I) 
    $$  

    Under a priori setting, we need to prove another helping claim that the optimal a priori master tour will always visit all vertices within the cluster $s(v)$ consecutively. This is true because starting from some vertex in $s(v)$, we cannot gain benefits from visiting another vertex in $s(v')$ next, instead of visiting the remaining vertices in $s(v)$ and then visit that vertex in $s(v')$, since going between vertices in $s(v)$ cost $0$. Then it is easy to see that any a priori master tour $T = (v_1, v_2, \cdots, v_{|V|})$ of $V$ has a corresponding a priori master tour $T'$ of $V'$ which firstly visits all vertices in $s(v_1)$, and then visit all vertices in $s(v_2)$ and so on. It is also easy to see the cost of any a priori master tour $T$ is the same as the corresponding  a priori master tour $T'$ since going between vertices in $s(v)$ cost $0$.
    
\end{proof}

\section{Estimate of Rounds Needed}\label{sec:gridexample}

We consider a new graph where vertices are also placed in a $k$ by $k$ grid where $k = \Delta^\Delta$ for some parameter $\Delta$. Instead of connecting all vertices from column $i$ to column $i + 1$, the $j$-th vertex in $i$-th column will only connect to vertices $j'$ in the $i+1$-th column only when under $\Delta$ base, $j'$ differs from  $j$  in at most $1$ coordinate. Here $\Delta$ is a parameter that we firstly set to be $\frac{\log k}{\log \log k}$.

We say two vertices are connected if the distance between them is the same as the (cyclic) distance of their underlying column. It is then easy to see that any active vertex in $i$-th column are connected to any vertex in $i'$-th column if $i' \ge i + \Delta$. We call a path to have $1$ round if it starts from $i$-th column, and after moving only by connecting edges (the edges connecting  two \emph{connected} vertices) it comes across $i$-th column again. Clearly each round costs $\Theta(k)$, we want to know how many rounds do we need to cover all active vertices, which gives an upper bound of optimal a posteriori cost.

Denote $M := \max_{i} \sum_{r=-\Delta}^{\Delta} X_{i+r}$, where $X_i$ represents the number of active vertices in $i$-th column. Then we claim that we need at most $M$ rounds to cover all vertices.

We prove the claim by contradiction. Suppose after $M$ rounds, there are still a vertex $v$ in column $j$ which is not covered. Then consider the interval $I_j$ consisting of columns $[j - \Delta, j + \Delta]$, there are no more than $M$ active vertices in $I_j$, so within the $M$ rounds, there exists at least one round $r$ such that no vertices from $I_j$ is touched (by pigeonhole principle). Then we can adjust the previous path by adding vertex $j$ into round $r$. This is always possible because in the round $r$ we can always find the last vertex before $I_j$ and the first vertex after $I_j$, we can insert vertex $v$ without breaking the connection rule since   any active vertex in $i$-th column are connected to any vertex in $i'$-th column if $i' \ge i + \Delta$. We can always perform this adjustment until there is no such vertex exists, then we have a contradiction.

We then need to bound $\E[M]$ and the optimal a posteriori cost would be bounded by $\E[M] \cdot k$.

Define the cyclic window length $m := 2\Delta + 1$.
Assume $X_1, \dots, X_k$ are independent and
$X_i \sim \textbf{Bin}(k, 1/k)$.
Define the cyclic window sums
$S_i := \sum_{r=0}^{m-1} X_{i+r}$
(indices mod~$k$).
The quantity of interest is $M := \max_{1 \le i \le k} S_i$.

By the trick discussed, we know for any $T$:

\begin{align*}
    \E[M] \le T + \sum_{i=1}^k \E[(S_i - T)^+]
\end{align*}

This is nice because we do not need independence here (we do not have independence since intervals $S_i$ might be overlapping) and by symmetry, we have:

\begin{align*}
    \E[M] \le T + k \E[(S_i - T)^+]
\end{align*}

Then we convert the $\E[(S_1 - T)^+]$ into a tail sum. Since $S_i$ is integer-valued variable, so $\E[(S_1 - T)^+] = \sum_{u=1}^{\infty} \Pr[S_1 \ge T + u]$. So we have:

\begin{align*}
    \E[M] \le T + k \sum_{u=1}^{\infty} \Pr[S_1 \ge T + u]
\end{align*}

We know $S_1 \sim \textbf{Bin}(mk, 1/k)$ with mean $m$ and for large $k$ it is approximated by $\textbf{Poi}(m)$. So let $Z \sim \textbf{Poi}(m)$ and then $\Pr[S_1 \ge t] \simeq \Pr[Z \ge t]$. And for Poisson distribution, we have the property that $\Pr[Z \ge t + 1] \le \frac{m}{t + 1}\Pr[Z \ge t]$. This implies that the tail decreases very fast (geometrically fast) when $t > m$. Suppose $T + 2 > m$, then $\Pr[S_1 \ge T + u] \le (\frac{m}{T + 2})^{u-1} \Pr[Z \ge T + 1]$. So $\sum_{u \ge 1} \Pr[Z \ge T + u] \le \frac{\Pr[Z \ge T + 1]}{1 - m/(T + 2)} \le \Pr[Z \ge T] \cdot \frac{m}{T + 1} \cdot \frac{1}{1 - m / (T + 2)}$.

So by approximation, $\E[M] \lesssim T + k \Pr[Z \ge T] \cdot \frac{m}{T + 1} \cdot \frac{1}{1 - m / (T + 2)}$. We can pick $T$ such that $k \Pr[Z \ge T] \simeq 1$, then this would be a good upper bound. To find the $T$, we use the fact that $\Pr[Z \ge T] \simeq \exp(-m h(T/m))$ where $h(x) = x \ln x - x + 1$. Solving the equation we have $m h(T/m) \simeq \ln k$. Since $\ln k / m = \Theta(\ln \ln k)$ and $h(x) \rightarrow x \ln x$ when $x \rightarrow \infty$, so

$$
    \frac{T}{m} \ln (\frac{T}{m}) \simeq \Theta(\log \log k)
$$

So we can represent $T$ with Lambert $W$ function:

$$
\frac{T}{m} = \Theta\left(\frac{\log \log k}{W(\log \log k)}\right)
$$

Thus we have:

$$
T = \Theta\left(\frac{\log k}{W(\log \log k)}\right) = \Theta\left(\frac{\log k}{\log \log \log k}\right)
$$

So we know $\E[M] = \Theta\left(\frac{\log k}{\log \log \log k}\right)$ which means the a posteriori optimal cost is bounded by $k \cdot \Theta\left(\frac{\log k}{\log \log \log k}\right)$.

\section{How To Create a Gap Between a post And adaptive}
The example graph shown in section \ref{sec:gridexample} has small a post optimal cost, however the optimal adaptive cost is also small. In fact, it is easy to give a policy which achieves the same bound.

So now the problem is that how to create a gap between adaptive and a post. The issue of this grid example is that we cannot punish adaptive policy when it picks a vertex to call. If we want to create a gap, we should be able to punish adaptive policy, i.e., when policy picks some vertex $v$, we need to be able to construct the following graph with remaining uncalled vertices that the policy's choice is globally a bad choice.

\section{Clairvoyance gap lower bound example and proof}
\begin{lemma}\label{lem:matching-subdivision-gap}
For every integer $n \ge 3$, there exists a symmetric stochastic TSP instance
$\mathcal{I}_n = (\{r\}\cup V,d,\{p_v\}_{v \in V})$ such that
\[
\optprior(\mathcal{I}_n)=3n
\qquad\text{and}\qquad
\optpost(\mathcal{I}_n)=\frac{5n}{2}.
\]
Consequently,
\[
\frac{\optprior(\mathcal{I}_n)}{\optpost(\mathcal{I}_n)}=\frac{6}{5},
\]
and hence the obliviousness gap is at least $6/5$.
\end{lemma}

\begin{proof}
Fix $n \ge 3$.  We define the instance $\mathcal{I}_n$ as follows.

\medskip
\noindent
\textbf{The metric.}
Let
\[
U:=\{u_1,\ldots,u_n,v_1,\ldots,v_n\},
\]
where we identify the depot with $u_1$, i.e.\ $u_1=r$.
Consider the graph $G_n$ on the vertex set
\[
U \cup \{x_1,\ldots,x_n\}
\]
obtained as follows:
\begin{itemize}
    \item for each $i\in[n]$, add the two edges $u_i x_i$ and $x_i v_i$, each of length $1$;
    \item for every two distinct vertices $z,z' \in U$ with $\{z,z'\}\neq \{u_i,v_i\}$ for all $i\in[n]$, add the edge $zz'$ of length $1$.
\end{itemize}
The graph is essentially a complete graph on $2n$ vertices and then we find a perfect mathcing(with $n$ edges) and subvide the edges in the matching.
Let $d$ be the shortest-path metric of $G_n$.  In particular,
\begin{align*}
d(u_i,v_i)&=2 \qquad &&\text{for every } i\in[n],\\
d(z,z')&=1 \qquad &&\text{for distinct } z,z' \in U \text{ with } \{z,z'\}\neq \{u_i,v_i\}\ \forall i,\\
d(x_i,z)&=2 \qquad &&\text{for every } i\in[n] \text{ and every } z\in U\setminus\{u_i,v_i\},\\
d(x_i,x_j)&=3 \qquad &&\text{for every distinct } i,j\in[n].
\end{align*}
Since $d$ is a shortest-path metric, it is symmetric and satisfies the triangle inequality.

\medskip
\noindent
\textbf{The activation probabilities.}
Every original vertex is always active:
\[
p_z=1 \qquad \text{for every } z\in U
\]
and every subdividing vertex is active independently with probability $1/2$:
\[
p_{x_i}=\frac12 \qquad \text{for every } i\in[n].
\]

\textbf{Computation of $\optpost(\mathcal{I}_n)$.}
Fix a realization $A\subseteq V$, and write $S:=S(A)$, where $S(A)$
is defined as $S(A):=\{i\in[n]:x_i\in A\}$.

We first prove the lower bound
\[
\opt(A)\ge 2n+|S|.
\]
Indeed, any tour visiting $\{r\}\cup A$ uses exactly $2n+|S|$ edges, and every edge of the metric has length at least $1$.

We now prove the matching upper bound.
Consider contracting the subdivided edges in $G$ if the corresponding subdividing vertex
is active and then delete the inactive subdividing vertices. We denote this graph
as $G_A$.
It is clear that when $G_A$ has a Hamiltonian cycle, we can also find a tour in 
$G$ visiting all active vertices using only edges of length $1$.
So now the goal is to show $G_A$ is Hamiltonian and we consider Dirac's condition.
We prove $\delta(G_A) \ge \frac{1}{2} |V(G_A)| = \frac{1}{2} (2n - |S|)$, where 
$\delta(G_A)$ denotes the minimum degree.
For contracted vertices, the degree is $|V(G_A)| - 1$; for non-contracted vertices,
the degree is $|V(G_A)| - 2$ since these vertices lose 1 more degree because of 
deletion of inactive subdividing vertices.
Then we know when $n \ge 4, |V(G_A) - 2| \ge \frac{1}{2} |V(G_A)$ always holds.
For $n = 3$, we only need to consider the case when $|S| = 3$. In this special 
case, the resulting graph $G_A$ is simply $K_3$ so obviously it is Hamiltonian.
Hence for all cases we can always find a tour visiting all realised vertices
which has total length equal to its number of edges, namely $2n+|S|$.
Therefore
\[
\opt(A)=2n+|S|.
\]

Taking expectation over the random set $A$, we obtain
\[
\optpost(\mathcal{I}_n)
=
\E_{A\sim\mu}[\opt(A)]
=
2n+\E_{A\sim\mu}[|S(A)|]
=
2n+\frac{n}{2}
=
\frac{5n}{2}.
\]

\textbf{Lower bound on $\optprior(\mathcal{I}_n)$.}
Let $T$ be an arbitrary master tour. Delete all vertices $x_1,\ldots,x_n$ from $T$.
The resulting tour $\pi$ visits exactly the original vertices in $U$.
Let
\[
m:=\bigl|\{i\in[n]: u_i \text{ and } v_i \text{ are consecutive in } \pi\}\bigr|.
\]
Since $\pi$ has exactly $2n$ edges, each of length $1$ or $2$, and the edges of length $2$ are precisely the matched edges $u_iv_i$, we have
\[
c(\pi)=2n+m.
\]

For each $t\in\{0,1,\ldots,n\}$, let $T_t$ be the random tour obtained from $T$ by keeping
$x_1,\ldots,x_t$ whenever they are active and deleting $x_{t+1},\ldots,x_n$.
Thus $T_0=\pi$, and $T_n=T\vert_A$.

For $i\in[n]$.
In the current tour $T_{i-1}$, the vertex $x_i$ would be inserted between two consecutive vertices,
say $p_i$ and $q_i$, if $x_i$ is active.
Hence the increment in tour length caused by revealing $x_i$ is
\[
\Delta_i=
\begin{cases}
d(p_i,x_i)+d(x_i,q_i)-d(p_i,q_i), & \text{if } x_i\in A,\\[0.5ex]
0, & \text{if } x_i\notin A.
\end{cases}
\]
Since $p_{x_i}=1/2$, we obtain
\[
\E[\Delta_i \mid x_1,\ldots,x_{i-1}]
=
\frac12\Bigl(d(p_i,x_i)+d(x_i,q_i)-d(p_i,q_i)\Bigr).
\]

We claim that
\[
d(p_i,x_i)+d(x_i,q_i)-d(p_i,q_i)=0
\quad\Longleftrightarrow\quad
\{p_i,q_i\}=\{u_i,v_i\},
\]
and otherwise
\[
d(p_i,x_i)+d(x_i,q_i)-d(p_i,q_i)\ge 2.
\]
This follows directly from the explicit distance values listed above:
the value is $1+1-2=0$ when $\{p_i,q_i\}=\{u_i,v_i\}$, while in every other case the minimum possible value is $2$.

Now suppose that $u_i$ and $v_i$ are \emph{not} consecutive in $\pi$.
Deleting or re-inserting vertices $x_j$ never changes the cyclic order of the original vertices.
Hence $x_i$ can never be inserted between $u_i$ and $v_i$, and therefore
\[
\{p_i,q_i\}\neq \{u_i,v_i\}.
\]
Thus we have:
\[
\E[\Delta_i \mid x_1,\ldots,x_{i-1}] \ge 1.
\]
Consequently,
\[
\E[\Delta_i]\ge 1
\qquad\text{for at least } n-m \text{ indices } i.
\]
For the remaining $m$ indices we use the trivial bound $\E[\Delta_i]\ge 0$.

Summing over $i\in[n]$, we obtain
\[
\E_{A\sim\mu}[c(T\vert_A)]
=
c(\pi)+\sum_{i=1}^n \E[\Delta_i]
\ge
(2n+m)+(n-m)
=
3n.
\]
Since $T$ was arbitrary, this shows that
\[
\optprior(\mathcal{I}_n)\ge 3n.
\]

Together with the upper bound of a posteriori optimimum, we conclude that
\[
\frac{\optprior(\mathcal{I}_n)}{\optpost(\mathcal{I}_n)}=\frac{3n}{5n/2}=\frac65.
\]
This completes the proof.
\end{proof}
