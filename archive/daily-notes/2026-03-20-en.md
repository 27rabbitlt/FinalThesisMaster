## Meeting with Mentor

Had a discussion with my mentor. Several tasks identified for future work:

### Task 1: Tighter bound for AdaptOpt
In the paper, AdaptOpt is bounded by Phi(2). Can we give a tighter bound? Or what happens if AdaptOpt is strictly less than Phi(2)?

### Task 2: Similarity between Adapt TSP and A Posteriori TSP solvers
Why does the adaptive TSP solver algorithm look so similar to the a posteriori TSP solver (both using state compression DP)? What takeaway message can we get from this structural similarity?

### Task 3: LP relaxation approach (inspired by stochastic knapsack)
In the stochastic knapsack paper, they bound the adaptivity gap using LP. Can we do something similar for TSP? What information can we extract from the LP?

### Task 4: Tighten the grid example analysis
Our current analysis of a posteriori in the grid example may be too loose. Try making the example smaller to get a tighter result.

### Task 5: Visualize large-gap examples
Try to visualize examples that create a relatively large gap between a posteriori and adaptive.
